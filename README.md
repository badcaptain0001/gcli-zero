# git-workflow-automator

Git workflow automation CLI tool.

## Installation

```bash
pip install -e .
```

## Usage

```bash
gwa --help

# Generate challenge evidence templates automatically
gwa init
# slash alias
gwa /init
```

`gwa init` creates:
- `COPILOT_LOG.md`
- `DEV_SUBMISSION_TEMPLATE.md`
